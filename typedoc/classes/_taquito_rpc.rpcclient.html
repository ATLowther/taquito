    example

    new RpcClient('https://mainnet.tezrpc.me', 'main') this will use https://mainnet.tezrpc.me/chains/main

    Parameters

    • Default value url: string = defaultRPC

      rpc root url (default https://mainnet.tezrpc.me)

    • Default value chain: string = defaultChain

      chain (default main)

    • Default value httpBackend: HttpBackend = new HttpBackend()
