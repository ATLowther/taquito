Taquito boilerplate

Framework agnostic demo#

Taquito boilerplate is a framework-agnostic starter kit for developing web based applications running on Tezos network.

For information about how to get started with taquito boilerplate please refer to the Getting Started section in the README found in the taquito-boilerplate repository in Github.

React demo#

You can also download a demo app written in React to try Taquito. The template for the app is available in the Taquito React template repo and only requires a few commands to be up and running.
