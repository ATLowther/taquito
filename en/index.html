A TypeScript library suite for development on the Tezos blockchain.

Quick Start

Easy to Use

Hit the ground running using & contributing to the library quickly: Taquito is written in an idiomatic TypeScript style, and includes a set of ready-made React components.

Portable

Perfect for any JavaScript project on the front- or back-end with minimal dependencies, Taquito has no reliance on any stack by default, except Tezos node.

Well-Supported

Taquito comes complete with a well-documented API using TypeDoc, nightly and continuous integration tests against the Nomadic Labs Tezos node & versioned releases published to npmjs.org.

Participation in CII Badging Program

The CII (Core Infrastructure Initiative) badging program was created by the Linux Foundation in response to previous security issues in open-source projects. We are committed to follow these best practices and earn/maintain our CII Badges.

Boilerplate App

The Taquito team has created a small sample project that checks and displays XTZ balance. Developers are invited to use this as a starting point by simply forking the Taquito Boilerplate project in GitHub.
